# image mode设置不显示或失效。

2021年6月30日

给 width 和 height 设置 inherit 继承一下即可。

主要是因为小程序 image 默认会给 width 或 height ，限制了 mode 效果。

--

# *登录失效

2021年7月12日

获取的session_key不正确，导致解密操作错误，返回pad block corrupted。但是导致session_key不正确的原因是第二步调用wx.login()方法，会刷新session_key。

处理办法：务必将 wx.login 在获取授权前执行获取 session_key，不能在授权按钮上绑定此事件，因为一旦点击获取授权，``wx.login` 的获取就会后刷新，与授权解密不一致。

wx.login -> wx.getPhoneNumber/getUserProfile （不可异步）

--

# 微信小程序 wxs 文件中使用正则 replace 字符串报错

2021年9月28日

values.replace(正则,01) 失效

处理办法：getRegExp()

**values.replace(getRegExp( 正则 , 'g' ) , 01 ) **

--

# 微信小程序 多行隐藏 ios 显示异常问题

2021年10月14日

display: -webkit-box;/*对象作为弹性盒子使用*/
-webkit-line-clamp: 2;/*不规则属性，只有webkit内核的浏览器才支持，移动设备大部分支持*/
-webkit-box-orient: vertical;/*设置或检索伸缩盒子对象的子元素的排列方式*/

处理办法：当前元素height不能限制！

--
