太灰2021-06-11 08:00:00技术拓展建站 vuepress theme

TIP

很喜欢 reco 的这款主题,在大佬的文件上做了微不足道的调整...

# 即拿即用

# 安装 vuepress

yarn global add vuepress@^1.8.2 # 或者：npm install -g vuepress

# 安装 vuepress-theme-reco 主题

yarn add vuepress-theme-reco@^1.6.6 # 或者：npm install vuepress-theme-reco --save-dev

# 下载并替换对应目录文件

下载地址：https://gitee.com/phygray/vuepress-reco-custom

// !! 注意替换文件前后注意留备份，如果加装新依赖会全部重新默认，需要再覆盖一次替换包。！！
// !! 请尽可能避免重载依赖 ！！

# 本地启动

yarn docs dev   或者：vuepress dev docs

# 由浅入深

主题
